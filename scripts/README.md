# Scripts

Node helpers invoked from **`package.json`** (not imported by the Vite app at runtime).

**Behavior, CI gates, limitations:** [docs/scripts/README.md](../docs/scripts/README.md) · **Command reference:** [AGENTS.md](../AGENTS.md) · **Quality / API artifacts:** [STANDARDS.md](../docs/STANDARDS.md).

| File                                                                       | Used by                                                   | Purpose                                                                                                                                                                                   |
| -------------------------------------------------------------------------- | --------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`cypress-e2e-shard.mjs`](./cypress-e2e-shard.mjs)                         | `pnpm run e2e:ci:shard`                                   | Splits `cypress/e2e/*` across `CYPRESS_SHARD` / `CYPRESS_SHARD_TOTAL`.                                                                                                                    |
| [`openapi-edge-functions.mjs`](./openapi-edge-functions.mjs)               | `pnpm run openapi:edge-functions`                         | Writes [`openapi/supabase-edge-functions.json`](../openapi/supabase-edge-functions.json).                                                                                                 |
| [`openapi-edge-functions-check.mjs`](./openapi-edge-functions-check.mjs)   | `pnpm run openapi:edge-functions:check`                   | Regenerate OpenAPI + fail on git diff.                                                                                                                                                    |
| [`build-postman-collection.mjs`](./build-postman-collection.mjs)           | `pnpm run postman:collection`                             | Writes [`postman/Lucid-Web-Craftsman.postman_collection.json`](../postman/Lucid-Web-Craftsman.postman_collection.json).                                                                   |
| [`postman-collection-check.mjs`](./postman-collection-check.mjs)           | `pnpm run postman:collection:check`                       | Regenerate Postman + fail on git diff.                                                                                                                                                    |
| [`deploy-all-supabase-functions.mjs`](./deploy-all-supabase-functions.mjs) | `pnpm run deploy:functions:all`                           | Sequentially deploys every `supabase/functions/<name>/index.ts` (skips `_shared`). Requires `supabase link` + login.                                                                      |
| [`check-edge-functions-bundling.mjs`](./check-edge-functions-bundling.mjs) | `pnpm run validate`, `check:edge-functions:bundling*`, CI | Blocks sibling Edge Function imports (use `_shared/`); optional `deno check`. Flags in script header (`--baseline`, `--report-*`).                                                        |
| [`lib/classify-api-layer.mjs`](./lib/classify-api-layer.mjs)               | `classify-api-failure`, `report-e2e-http-failures`        | Parses URL/path → mock / PostgREST / Edge **label + hint** (no I/O); see [PLATFORM.md § diagnosing](../docs/PLATFORM.md#diagnosing-api-and-database-failures).                            |
| [`classify-api-failure.mjs`](./classify-api-failure.mjs)                   | ad hoc                                                    | CLI: `node scripts/classify-api-failure.mjs "<url-or-path>"` (or stdin).                                                                                                                  |
| [`report-e2e-http-failures.mjs`](./report-e2e-http-failures.mjs)           | manual or optional E2E wrapper                            | Reads `cypress/diagnostics/http-failures.json` from [`networkFailureCapture.ts`](../cypress/support/networkFailureCapture.ts); not wired in default **`package.json`** `e2e:ci*` scripts. |

`pnpm run api:artifacts` runs the OpenAPI generator then the Postman builder. See [`openapi/README.md`](../openapi/README.md) and [`postman/README.md`](../postman/README.md).
