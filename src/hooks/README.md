# `src/hooks/`

Feature and infrastructure hooks for the SPA. The maintained catalog — purpose, dependencies, and test status for every hook in this folder (and the related hooks under `src/stores`, `src/context`, `src/lib`, `src/components/performance`, `src/utils`) — lives at:

- [`docs/HOOKS.md`](../../docs/HOOKS.md)

That doc also contains the primer on when to use hooks, how to test them with `renderHook`, and how to write accurate mocks for this codebase.

When adding a new hook here, update the relevant table in [`docs/HOOKS.md`](../../docs/HOOKS.md) in the same PR.
